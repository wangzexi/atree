# AGENTS.md

This repository is the next-generation A-Tree product experiment.

Keep the first implementation small and centered on the product core:

- left-side autonomous node tree
- right-side directory-bound AI conversation
- directory sessions
- loops or scheduled sessions
- interface agents for autonomous directories

Do not start with a full file previewer or graph UI. The physical file tree is
the storage substrate; the first UI should expose the operational node tree.
